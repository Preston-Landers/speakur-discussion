An example of using <speakur-discussion>:

This page represents some web content such as an article, a blog post, or any other resource with a URL. It can be a either a static resource or a place within a dynamic web app.

The reason this page uses Comic Sans font is mainly to test that Speakur is largely independent of the page style.

Inside the page we wish to embed a discussion forum so users can post their reactions, feedback, etc. That is where Speakur comes in. It provides an easily embeddable discussion forum by providing a custom HTML tag that can be inserted anywhere in the page.

Here's an example of how it works:

  • You have a blog or website.
  • You create a Firebase app to host the comment data.
  • You add speakur-discussion and its dependencies to your bower_components directory.
  • You add the <speakur-discussion> element to your page where you want it to show up, and point the element at your Firebase URL.
  • You now have a full-featured commenting system on your site!
